# 新设备同步

安装好 node 和 git，并配置好 git 访问 github 仓库，然后克隆 hexo 分支到本地

```git
git clone -b hexo git@github.com:<your_username>/<your_username>.github.io
```

然后进入 `<your_username>.github.io` 目录

```git
npm install
```
